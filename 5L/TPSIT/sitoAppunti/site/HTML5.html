Introduzione

HTML (HyperText Markup Language) è il linguaggio di markup standard utilizzato per creare pagine web. Con l'aiuto dei tag e degli attributi, HTML definisce la struttura e la disposizione del contenuto di una pagina web. I browser interpretano il codice HTML per visualizzare il testo, le immagini, i link e altri elementi multimediali nel modo desiderato dagli sviluppatori web.

Inizio

Ogni documento HTML inizia con una dichiarazione di tipo documento e l'elemento radice <html> . Il tag <html> contiene due sezioni principali: <head> e <body> . Nella sezione <head> , puoi inserire i metadati, come il titolo della pagina ei collegamenti a file CSS e script JavaScript. La sezione <body> conterrà tutto il contenuto principale della pagina, visibile nell'area di visualizzazione del browser.
N.B. Installando le estensioni di Visual Studio Code, è possibile utilizzare il comando html:5 per scrivere in automatico la base dell'HTML
Ecco un esempio di un documento HTML di base:

              <!DOCTYPE html>

              <html>

              <head>

              <meta charset="UTF-8">

                <title>Titolo della Pagina</title>

                <!-- Inserisci qui eventuali collegamenti a file CSS e script JavaScript -->

              </head>

              <body>

                <!-- Contenuto visibile nella pagina -->

                <h1>Questo è un titolo di primo livello</h1>

                <p>Questo è un paragrafo.</p>

                <img src="immagine.jpg" alt="Descrizione immagine">

                <a href="https://www.esempio.com">Link ad esempio</a>

              </body>

              </html>

Collegare CSS, JS

Per collegare CSS e file/script JavaScript esterni alla tua pagina HTML, puoi utilizzare gli elementi <link> e <script> all'interno della sezione <head> del documento. Questo è utile per mantenere il codice HTML pulito e separare la struttura del contenuto (HTML) dalla sua presentazione (CSS) e logica interattiva (JavaScript)
N.B. I file devono essere contenuti all'interno della stessa cartella, altrimenti bisogna scrivere il percorso per accedere al file.
Esempio di collegamento a un file CSS:

                <head>
                  <link rel="stylesheet" href="stile.css">
                </head>
Esempio di collegamento a un file JavaScript:

                <head>
                  <script src="script.js""></script>
                </head>

Immagini

Per inserire un'immagine nella pagina HTML, inserire l'elemento <img> . L'attributo "src" specifica il percorso dell'immagine e l'attributo "alt" fornisce una descrizione alternativa dell'immagine, che è utile per l'accessibilità.
Esempio di inserimento di un'immagine: 

                  <img src="immagine.jpg" alt="Descrizione dell'immagine">

Collegamenti ipertestuali

Per creare collegamenti ipertestuali (link) tra diverse pagine web o risorse, utilizzare l'elemento <a> . L'attributo "href" specifica l'URL di destinazione a cui punta il link.
Esempio di collegamento ipertestuale:

                  <a href="https://www.google.com">Visita il sito di Google</a>
Visita il sito di Google

Elenchi

HTML supporta due tipi di elenchi: elenchi non ordinati ( <ul> ) e elenchi ordinati ( <ol> ). Gli elementi dell'elenco sono definiti con il tag <li> . Esempio di elenco non ordinato:

                  <ul> / <ol>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                    <li>Elemento 3</li>
                  </ul> / </ol>

  • Elemento 2
  • Elemento 3

Tabelle

Per creare tabelle in HTML, utilizzare gli elementi <table> , <tr> (righe), <th> (intestazioni di colonna) e <td> (dati delle celle).
Esempio di tabella semplice:

                  <table>
                    <tr>
                      <th>Nome</th>
                      <th>Cognome</th>
                      <th>Età</th>
                    </tr>
                    <tr>
                      <td>Mario</td>
                      <td>Rossi</td>
                      <td>30</td>
                    </tr>
                    <tr>
                      <td>Laura</td>
                      <td>Bianchi</td>
                      <td>25</td>
                    </tr>
                  </table>

Titoli

Gli elementi di titolo in HTML sono utilizzati per creare titoli e sottotitoli all'interno della pagina. Essi vanno dal livello 1 al livello 6, dove <h1> è il titolo di primo livello e <h6> è il titolo di sesto livello. Generalmente, dovresti utilizzare <h1> per il titolo principale della pagina e utilizzare i livelli successivi per i sottotitoli.
Esempio:

                  <h1>Questo è un titolo di primo livello</h1>
                  <h2>Questo è un titolo di secondo livello</h2>
                  <h3>Questo è un titolo di terzo livello</h3>
                  <h4>Questo è un titolo di quarto livello</h4>
                  <h5>Questo è un titolo di quinto livello</h5>
                  <h6>Questo è un titolo di sesto livello</h6>

Testo

Gli elementi di testo in HTML sono utilizzati per formattare il contenuto del testo all'interno della pagina. Alcuni degli elementi più comuni includono: <p> : L'elemento di paragrafo crea blocchi di testo. È utilizzato per separare i commenti nel documento.

                  <p>Questo è un paragrafo. Il testo all'interno di questo elemento verrà visualizzato come un blocco di testo.</p>

Questo è un paragrafo. Il testo all'interno di questo elemento verrà visualizzato come un blocco di testo.

<strong> : L'elemento forte è utilizzato per evidenziare il testo in grassetto, che indica solitamente una maggiore importanza o enfasi. 
                  <p>Questo è un testo <strong>in grassetto</strong>.</p>

Questo è un testo in grassetto.

<em> : L'elemento em è utilizzato per evidenziare il testo in corsivo, che indica una lieve enfasi o importanza. 
                  <p>Questo è un testo <em>in corsivo</em>.</p>

Questo è un testo in corsivo.

<mark> : L'elemento mark viene utilizzato per evidenziare parti specifiche del testo, ad esempio per scopi di riferimento o sottolineatura. 
                  <p>Questo è un testo <mark>evidenziato</mark> per attirare l'attenzione.</p>

Questo è un testo evidenziato per attirare l'attenzione.

<del> : L'elemento del viene utilizzato per mostrare testo barrato, comunemente usato per indicare del testo cancellato o obsoleto. 
                  <p>Questo è un testo <del>barrato</del>.</p>

Questo è un testo barrato.

<ins> : L'elemento ins viene utilizzato per mostrare testo sottolineato, che indica l'aggiunta di testo o una modifica recente. 
                  <p>Questo è un testo <ins>sottolineato>/ins>.</p>

Questo è un testo sottolineato.

Section

l tag <section>è un elemento semantico utilizzato per rappresentare una sezione di contenuto all'interno di un documento HTML. È uno dei numerosi elementi semantici introdotti in HTML5 per fornire una struttura e un significato più chiaro al contenuto di una pagina web.
N.B. Al posto di "section" è possibile scrivere quello che meglio di crede, per esempio di può utilizzare il <footer> se si tratta di una sezione riservata al footer, oppure <contats> se è una sezione dei contatti.
Esempio:

                  <section>
                    <!-- Contenuto della sezione -->
                  </section>

Favicon

Il favicon è un'elemento estetico che troviamo all'interno della scheda del nostro browser, ovvero l'immagine che troviamo alla sinistra del nome della pagina in cui siamo. Per aggiungere una favicon al tuo sito web, salva l'immagine della tua favicon nella directory principale del tuo server web oppure crea una cartella nella directory principale chiamata images e salva l'immagine della tua favicon in questa cartella. Un nome comune per un'immagine favicon è "favicon.ico". Successivamente, aggiungi un <link>elemento al tuo file "index.html", dopo l' <title>elemento.
Esempio:

                  <head>
                    <title>My Page Title</title>
                    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
                  </head>

ID / Classi

L' id attributo HTML viene utilizzato per specificare un ID univoco per un elemento HTML. Non puoi avere più di un elemento con lo stesso id in un documento HTML; invece se si vuole applicare uno stile oppure fare un gruppo di div che si comportino allo stesso modo, basta inserire class al posto di id.
Esempio:

                  <h1 id="myHeader">My Header</h1>
                  <h1 class="myHeader">My Header</h1>

Formattazione

All'interno dell'HTML sono presenti alcuni comandi per effettuare della formattazione su alcune porzioni di testo, tramite questo comandi: <b>- Testo grassetto
<strong>- Testo importante
<i>- Testo in corsivo
<em>- Testo sottolineato
<mark>- Testo contrassegnato
<small>- Testo più piccolo
<del>- Testo cancellato
<ins>- Testo inserito
<sub>- Testo in pedice
<sup>- Testo in apice

Esempio:

                  <b>Questo testo è in grassetto</b>
                  <strong>Questo testo è importante</strong>
                  <i>Questo testo è corsivo</i>
                  <em>Questo testo è enfatizzaato</em>
                  <small>Questo testo è piccolo.</small>

                  <p>Il mio colore preferito è il  <del>blu</del> rosso.</p>
