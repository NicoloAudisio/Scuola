CSS è un linguaggio che gestisce il design e la presentazione delle pagine web

Introduzione

Il CSS, acronimo di Cascading Style Sheets, serve principalmente a definire l'aspetto e la formattazione delle pagine web. In altre parole, il CSS è utilizzato per controllare come il contenuto HTML viene presentato e visualizzato sullo schermo o sulla stampa.

Sintassi

Il CSS ha una sintassi molto semplice e intuitiva, composta da un selettore e dalle dichiarazioni.
Il selettore è il nostro ID oppure della nostra classe che noi andiamo ad inserire nel nostro codice HTML, se non andiamo ad inserire un nominativo, ci basta utilizzare il modulo HTML utilizzato, come può essere l'h1 oppure il p.
Ecco un esempio di CSS per modificare l'h1:

              h1 {
                color: red;
                text-align: center;
              }

Selettori

Il selettore id utilizza l'attributo id di un elemento HTML per selezionare un elemento specifico. Per selezionare un elemento con un id specifico, scrivi un cancelletto (#), seguito dall'id dell'elemento, invece se si utilizza la classe all'interno dell'HTML si utilizza il punto (.) prima del nome della classe.
N.B. Se si vuole modificare un componente contenuto all'interno di una classe, bisogna prima inserire la classe e poi il selettore
Esempio di utilizzo di un selettore id:

                #intro {
                  text-align: right;
                  color: black;
                }

Esempio di utilizzo di un selettore classe:

                .footer {
                  text-align: center;
                  font-size: 30px;
                }

Esempio di utilizzo di un selettore all'interno di una classe:

                .box h2 {
                  margin: 0 0 30px;
                  padding: 0px;
                  color: #fff;
                  text-align: center;
                }

Commenti

I commenti vengono utilizzati per spiegare pezzi di codice oppure per scrivere degli appunti o dei promemoria.
Esempio per l'utilizzo di un commento:

                  /* STILE DELLA NAVBAR */
                  .navbar {
                    background-color: white;
                    color: black;
                    text-align: center;
                  }

Colori

Il CSS consente di modificare il colore di ogni componente che viene inserito all'interno dell'HTML, si utilizzano i nomi e i colori predefiniti oppure i valori RGB, HEX, HSL, RGBA, HSLA; HTML e CSS supporta 140 nomi di colori standard. Esempio di cambio colore:

                  /* Cambio colore h1 */
                  h1 {
                    color: red
                  }

Sfondi

Lo sfondo unito con il CSS viene utilizzato per realizzare effetti grafici agli sfondi che andiamo ad inserire nel nostro codice.
Con il CSS possiamo utilizzare i seguenti comandi:
○ background-color -> Serve per modificare il colore del nostro sfondo.
○ background-image -> Serve per impostare o modificare un'immagine di sfondo.
○ background-repeat -> Serve per impostare la ripetizione dello sfondo.
○ background-attachment -> Serve per impostare la dinamicità dello sfondo tramite i comandi scroll per dare la possibilità di muovere tramite la rotella del mouse lo sfondo in verticale; fixed per fissare lo sfondo; local per far muovere lo sfondo con lo scrollio della pagina ed infine tutti questi comandi possono essere combinati tra di loro.
○ background-position -> Serve per impostare la posizione del nostro sfondo.

Bordi

I bordi servono per delineare graficamente le dimensioni di un <div> e possono essere utilizzati anche come separatori tra due o più argomenti.
Tramite il border-style è possibile modificare la grafica di questi bordi: ○ dotted -> Definisce un bordo punteggiato.
○ dashed -> Definisce un bordo tratteggiato.
○ solid -> Definisce un bordo solido.
○ double -> Definisce un doppio bordo.
○ groove -> Definisce un bordo scanalato 3D. L'effetto dipende dal valore del colore del bordo.
○ ridge -> Definisce un bordo increspato 3D. L'effetto dipende dal valore del colore del bordo.
○ inset -> Definisce un bordo interno 3D. L'effetto dipende dal valore del colore del bordo.
○ outset -> Definisce un bordo iniziale 3D. L'effetto dipende dal valore del colore del bordo.
○ none -> Non definisce alcun confine.
○ hidden -> Definisce un bordo nascosto.

Esempio:

                  <h1>Questo è un titolo di primo livello</h1>
                  <h2>Questo è un titolo di secondo livello</h2>
                  <h3>Questo è un titolo di terzo livello</h3>
                  <h4>Questo è un titolo di quarto livello</h4>
                  <h5>Questo è un titolo di quinto livello</h5>
                  <h6>Questo è un titolo di sesto livello</h6>

Testo

Gli elementi di testo in HTML sono utilizzati per formattare il contenuto del testo all'interno della pagina. Alcuni degli elementi più comuni includono: <p> : L'elemento di paragrafo crea blocchi di testo. È utilizzato per separare i commenti nel documento.

                  <p>Questo è un paragrafo. Il testo all'interno di questo elemento verrà visualizzato come un blocco di testo.</p>

Questo è un paragrafo. Il testo all'interno di questo elemento verrà visualizzato come un blocco di testo.

<strong> : L'elemento forte è utilizzato per evidenziare il testo in grassetto, che indica solitamente una maggiore importanza o enfasi. 
                  <p>Questo è un testo <strong>in grassetto</strong>.</p>

Questo è un testo in grassetto.

<em> : L'elemento em è utilizzato per evidenziare il testo in corsivo, che indica una lieve enfasi o importanza. 
                  <p>Questo è un testo <em>in corsivo</em>.</p>

Questo è un testo in corsivo.

<mark> : L'elemento mark viene utilizzato per evidenziare parti specifiche del testo, ad esempio per scopi di riferimento o sottolineatura. 
                  <p>Questo è un testo <mark>evidenziato</mark> per attirare l'attenzione.</p>

Questo è un testo evidenziato per attirare l'attenzione.

<del> : L'elemento del viene utilizzato per mostrare testo barrato, comunemente usato per indicare del testo cancellato o obsoleto. 
                  <p>Questo è un testo <del>barrato</del>.</p>

Questo è un testo barrato.

<ins> : L'elemento ins viene utilizzato per mostrare testo sottolineato, che indica l'aggiunta di testo o una modifica recente. 
                  <p>Questo è un testo <ins>sottolineato>/ins>.</p>

Questo è un testo sottolineato.

Section

l tag <section>è un elemento semantico utilizzato per rappresentare una sezione di contenuto all'interno di un documento HTML. È uno dei numerosi elementi semantici introdotti in HTML5 per fornire una struttura e un significato più chiaro al contenuto di una pagina web.
N.B. Al posto di "section" è possibile scrivere quello che meglio di crede, per esempio di può utilizzare il <footer> se si tratta di una sezione riservata al footer, oppure <contats> se è una sezione dei contatti.
Esempio:

                  <section>
                    <!-- Contenuto della sezione -->
                  </section>

Favicon

Il favicon è un'elemento estetico che troviamo all'interno della scheda del nostro browser, ovvero l'immagine che troviamo alla sinistra del nome della pagina in cui siamo. Per aggiungere una favicon al tuo sito web, salva l'immagine della tua favicon nella directory principale del tuo server web oppure crea una cartella nella directory principale chiamata images e salva l'immagine della tua favicon in questa cartella. Un nome comune per un'immagine favicon è "favicon.ico". Successivamente, aggiungi un <link>elemento al tuo file "index.html", dopo l' <title>elemento.
Esempio:

                  <head>
                    <title>My Page Title</title>
                    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
                  </head>

ID / Classi

L' id attributo HTML viene utilizzato per specificare un ID univoco per un elemento HTML. Non puoi avere più di un elemento con lo stesso id in un documento HTML; invece se si vuole applicare uno stile oppure fare un gruppo di div che si comportino allo stesso modo, basta inserire class al posto di id.
Esempio:

                  <h1 id="myHeader">My Header</h1>
                  <h1 class="myHeader">My Header</h1>

Formattazione

<i>- Testo in corsivo
